# Github-dashboard

### 套件

- antd([Ant Design](https://ant.design))
- axios
- react v.16.4.2
- react-router-dom  v.4.3.1

### API

github API

### 功能

可以搜尋Github上所有使用者或專案
可以觀看自己所有的repo、follower、folloeing、
預設是顯示自己的github內如

### 開啟專案

```javascript=
yarn install
yarn start
```
